# HMS – Dra Joyce Radis

Repositório exclusivo para o **gerador de evolução / reavaliação / passagem** do PS do Hospital Meridional Serra.

**Arquivo principal:** `index.html`

> HTML + CSS + JavaScript standalone — roda direto no navegador, sem backend nem dependências.

### Como abrir

1. Clique em **Code ▾ → Download ZIP** (ou clone o repo).
2. Abra `index.html` no navegador.

Quando o GitHub Pages estiver habilitado, bastará visitar:

```
https://drajoyceradis.github.io/HMS-Dra-Joyce-Radis/
```

### Aviso de privacidade

Não inserir nome, CPF, SAME, carteirinha, data de nascimento completa, chave de acesso ou qualquer identificador de paciente. A ferramenta roda localmente (offline) e não envia dados a servidor.
